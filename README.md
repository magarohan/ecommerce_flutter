# 🛍️ Ecommerce Flutter App

A modern Flutter-based E-commerce mobile application with clean UI, product browsing, cart management, and seamless shopping experience for Android and iOS.

---

# ✨ Features

## 👤 User Features

* User Authentication
* Product Listing
* Product Categories
* Product Details Screen
* Add to Cart
* Wishlist Support
* Search Products
* Checkout Flow
* Order Summary
* Responsive UI
* Dark/Light Theme Support

# 🚀 Getting Started

## Prerequisites

Install the following:

* Flutter SDK
* Android Studio / VS Code
* Android Emulator or Physical Device
* Git

Check Flutter installation:

```bash id="e6d4x1"
flutter doctor
```

---

# 📥 Installation

## Clone the repository

```bash id="g7f8k2"
git clone https://github.com/magarohan/ecommerce_flutter.git
cd ecommerce_flutter
```

## Install dependencies

```bash id="m2x9c7"
flutter pub get
```

---

# 🔑 Environment Setup

Create a `.env` file in the root directory:

```env id="b1w5l9"
API_BASE_URL=
FIREBASE_API_KEY=
GOOGLE_MAPS_API_KEY=
STRIPE_PUBLIC_KEY=
```

---

# ▶️ Run the App

```bash id="r4t8y1"
flutter run
```

---

# 🏗️ Build APK

## Android

```bash id="u7i2o3"
flutter build apk --release
```

## iOS

```bash id="v9k4n6"
flutter build ios --release
```
